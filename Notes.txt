State Management :

- Preserve the state of the web page ,control,data 

Two ways :
1.Client side SM : happens on the browser / client computer - cookie,View state
2.Server side SM : stores on the server - session

Why :

HTTP - Hyper text transfer protocol 

HTTP is a stateless protocol

- Responsibility to maintain the state in the application 

cookies :

- Small piece of information that is stored on the browser

View State :

- Transfer the data from controller actions to the browser

* ViewBag - complex data 
* View Data - simple data
* Temp Data - simple data
