HTML - Hyper text markup language
- creating the web page
- define the structure of the web page 
- standard markup language 
- consist of series of elements
- html is not case sensitive


HTML Structure :

<!DOCTYPE html> - defines the document in the html5 document
<html> - root element 
<head> - contains meta information
<title> - specify the title for your web page
<body> - defines the html document body and act as container to hold other html elements
<h1> ....<h6> : header tag
<p>- paragraph
